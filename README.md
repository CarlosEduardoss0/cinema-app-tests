# cinema-app-tests

Este repositório contém a aplicação de cinema (ou o projeto base) e a suíte de testes correspondente. O foco é garantir a qualidade e a Experiência do Usuário (UX) em todas as funcionalidades do aplicativo de gerenciamento e visualização de filmes/sessões.

## Visão Geral do Projeto

O objetivo principal deste projeto é desenvolver uma aplicação robusta e amigável para usuários que desejam consultar a programação de cinema, fazer reservas ou visualizar informações de filmes.

A pasta principal (`cinema-app-tests`) é dedicada à manutenção dos testes de usabilidade, testes automatizados e componentes de interface que serão avaliados.

## Como Executar o Projeto Localmente

### Pré-requisitos

* Node.js (versão recomendada para o seu projeto)
* npm ou yarn
* Git

### Instalação e Execução

1.  **Clone o Repositório:**
    ```bash
    git clone [https://github.com/CarlosEduardoss0/cinema-app-tests.git](https://github.com/CarlosEduardoss0/cinema-app-tests.git)
    cd cinema-app-tests
    ```

2.  **Instale as Dependências:**
    ```bash
    npm install
    # ou
    yarn install
    ```

3.  **Execute a Aplicação (se aplicável):**
    ```bash
    npm start
    ```
